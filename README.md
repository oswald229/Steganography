# Steganography

_Project built using Python 3 ._
#### Basic rules
- Pictures must be in `png` format.

#### Set up virtual environnment
`python3 -m venv env`
##### Linux & macOs
`source env/bin/activate`
##### Windows
`cd env/Scripts && activate`

##### Install needed packages

`pip intall -r requirements.txt`
#### Run the program
`python main.py`
